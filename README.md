\# 🧠 Adaptive Social Insight  
### Real-Time Opinion Mining of Digital Posts

A web-based application that analyzes user-generated content and classifies sentiment in real time. The system supports multilingual input, sarcasm-aware predictions, and basic image-based opinion analysis, integrated into an interactive web interface.

---

## 🚀 Features

- 💬 **Text-Based Sentiment Analysis**
  - Analyze input text and classify sentiment (Positive / Negative / Neutral)

- 😏 **Sarcasm-Aware Prediction**
  - Handles certain sarcastic expressions to improve sentiment accuracy

- 🌐 **Multilingual Support**
  - Supports multiple languages including:
    - Tamil  
    - Malayalam  
    - English  

- 🖼️ **Image-Based Opinion Analysis**
  - Accepts image input and extracts textual/contextual information for sentiment prediction

- ⚡ **Real-Time Prediction**
  - Fast processing and instant output display

- 🖥️ **Interactive UI**
  - Clean and responsive user interface built for ease of use

- 🔗 **Model Integration**
  - Integrated with a pre-trained sentiment analysis model
